Styling native `<input type="checkbox"/>` tag is difficult - this component mirrors a preceding input tag, to mimic and replace it in the UI.  
Use `<HiddenInput />` to create the interactive and invisible input.

Default styles include the following attributes:
- checked / unchecked
- disabled
- focus

Uses the following colors:
- --separator-color - unchecked border.
- --border-color - disabled unchecked 
- --muted-color - focus outline
- --base-color - checked
- --base-highlight - checked hover
- --muted-highlight - disabled checked
