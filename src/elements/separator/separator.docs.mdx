import { Separator } from './separator';

### Overview
Use these implicit CSS variables to style:  
`--separator-color`  
<br />

Separator is horizontal by default:
```js live=true
<div>
	top
	<Separator />
	bottom
</div>
```

Set separator to be vertical:
```js live=true
<div
	style={{
		width: 200,
		height: 40,
		display: 'flex',
		flexDirection: 'row',
		justifyContent: 'space-evenly',
	}}
	>
	left
		<Separator layout="vertical" />
	right
</div>
```
