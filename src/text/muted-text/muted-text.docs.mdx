### Overview
Text with muted (gray) color.  
Behaves like a regular *span* element.  
Applies text color with css variable `--text-muted`  

<br />

Pure css class name `.mutedText`, colors text as muted (gray).  
Use this directly as a class name without creating a DOM element.
